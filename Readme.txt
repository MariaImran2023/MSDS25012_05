# Diffusion Model for Image Generation

## Project Overview
This assignment implements a Diffusion Model for image generation using PyTorch. The model learns to generate images by reversing a diffusion process that gradually adds noise to images.

## Requirements
- Python 3.8+
- PyTorch 1.9+
- torchvision
- matplotlib
- numpy
- pillow
- tqdm

Install dependencies:
```bash
pip install torch torchvision matplotlib numpy pillow tqdm
